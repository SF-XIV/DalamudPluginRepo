# SF-XIV Dalamud Plugin Repository


## Installation

1. First open the Dalamud Settings menu in the game, accessible through typing `/xlsettings` in the chat or through the button at the bottom of the Plugin Installer.

2. Paste the following URL into the empty box at the bottom of the "Custom Plugin Repositories" section (the last box will always have a + next to it).

```
https://raw.githubusercontent.com/SF-XIV/DalamudPluginRepo/master/pluginmaster.json
```
